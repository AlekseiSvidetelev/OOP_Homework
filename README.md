# Описание проекта
Реализация классов для управления продуктами и категориями. Код включает модуль src/main.py, содержащий классы 
Product и Category, а также тесты в модуле tests/test_main.py.

## Установка и настройка
Для установки зависимостей и настройки проекта выполните следующие действия:

-poetry install

# Классы
## Класс Product
Класс Product представляет продукт и содержит следующие атрибуты:

name (str): Название продукта.
description (str): Описание продукта.
price (float): Цена продукта.
quantity (int): Количество продукта.


## Класс Category
Класс Category представляет категорию продуктов и содержит следующие атрибуты:

name (str): Название категории.
description (str): Описание категории.
products (list): Список продуктов в категории.
category_count (int): Счетчик созданных категорий.
product_count (int): Счетчик количества продуктов.


# Тесты
Тесты написаны с использованием библиотеки pytest и находятся в модуле tests/test_main.py.

# Фикстуры:
test_case_product(): Создает экземпляр класса Product для тестирования.
test_case_category(): Создает экземпляр класса Category для тестирования.

Тесты:
test_init_product(test_case_product):

Проверяет корректность инициализации объекта Product.

test_init_category(test_case_category):

Проверяет корректность инициализации объекта Category.
