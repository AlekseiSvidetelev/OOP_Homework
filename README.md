# Описание проекта
Реализация классов для управления продуктами и категориями. Код включает модуль src/main.py, содержащий классы 
Product и Category, а также тесты в модуле tests/test_main.py.

## Установка и настройка
Для установки зависимостей и настройки проекта выполните следующие действия:

-poetry install

# Классы

## Класс BaseProduct
Родительским для класса продуктов.

name (str): Название продукта.
description (str): Описание продукта.
price (float): Цена продукта.
quantity (int): Количество продукта.

## Класс-миксин MixinLog
При создании объекта, то есть при работе метода "__init__", печатает в консоль информацию о том, 
от какого класса и с какими параметрами был создан объект.

Например:

Product('Продукт1', 'Описание продукта', 1200, 10)

## Класс Product
Класс Product представляет продукт и содержит следующие атрибуты:

name (str): Название продукта.
description (str): Описание продукта.
price (float): Цена продукта.
quantity (int): Количество продукта.

Для класса Product реализован класс-метод new_product, который принимает параметры товара в виде словаря
и возвращает объект класса Product.
Для класса Product атрибут цены сделан приватным (__price). Реализованы геттер и сеттер для цены. В сеттере добавлена
проверка: если цена равна или ниже нуля, выводится сообщение в консоль, и цена не изменяется


## Класс Category
Класс Category представляет категорию продуктов и содержит следующие атрибуты:

name (str): Название категории.
description (str): Описание категории.
products (list): Список продуктов в категории сделан приватным атрибутом (__products), чтобы к нему нельзя было 
получить доступ извне. 

category_count (int): Счетчик созданных категорий.
product_count (int): Счетчик количества продуктов.

Для добавления товаров в категорию реализован метод add_product, который принимает объект класса Product
и добавляет его в приватный список.
Для возврата возможности просмотра товаров реализован геттер products, который возвращает список товаров в 
виде строки в формате: `Название продукта, 80 руб. Остаток: 15 шт.`


## Класс Smartphone
Класс «Смартфон» (Smartphone) расширяет класс Product атрибутами : 
производительность (efficiency),
модель (model), 
объем встроенной памяти (memory), 
цвет (color).


## Класс LawnGrass
Класс "LawnGrass" расширяет класс Product атрибутами: 
страна-производитель (country), 
срок прорастания (germination_period), 
цвет (color).
